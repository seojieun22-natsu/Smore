# 팝업스토어 운영 가이드

> SMORE 팝업스토어 운영 매뉴얼입니다. 모든 스텝은 근무 전 반드시 숙지해 주세요.

---

## 1. 근무 시간 및 시프트

### 운영시간

팝업마다 상이하며, 아래는 기본 구조입니다.

### 시프트 구조

| 시프트 | 시간 (예시) | 비고 |
|--------|------------|------|
| 오픈 | 08:30 ~ 13:30 / 09:00 ~ 14:00 | 2명 교대 |
| 미들 | 11:00 ~ 16:00 | |
| 마감 | 15:30 ~ 20:30 | |

### 휴게 시간

- 근무 중 30분 휴게
- 휴게 전후 **단톡방에 시간 보고** 필수
- 휴게 시간은 시프트별로 지정 (매니저 안내 참고)

---

## 2. 출퇴근 규정

### 출퇴근 인증

- **출근 시:** 단톡방에 "출근했습니다" 메시지
- **퇴근 시:** 퇴근 브리핑 작성 후 "퇴근했습니다" 메시지

### 휴게 보고

- **휴게 전:** "00:00~00:00 휴게 다녀오겠습니다"
- **휴게 후:** "00:00~00:00 휴게 다녀왔습니다"

### 근태 관련 문의

- 스케줄 변경/이슈 발생 시 **매니저에게 먼저 연락**
- 매니저가 본사 담당자에게 전달

---

## 3. 직원 복장 규정

### 필수사항

- 유니폼(맨투맨) 착용
- 스텝 목걸이 착용
- 단정한 머리 및 복장

### 금지사항

- 짧은 바지 금지
- 슬리퍼 금지
- 모자 + 마스크 동시 착용 금지

---

## 4. 오픈 스텝 업무

### 오픈 체크리스트

- [ ] 포스기 ON (전체)
- [ ] 조명 ON
- [ ] 배경 음악 ON
- [ ] AI 사진기 ON (해당 시)
- [ ] 토이카메라 샘플 배치 및 배터리 확인
- [ ] 매장 청소 (먼지, 쓰레기통 비닐)
- [ ] 매대 상품 진열 상태 확인
- [ ] 하부장 재고 보충
- [ ] 샘플 상품 전체 확인 (분실/훼손 체크)
- [ ] 오픈 전 매장 사진 촬영 → 단톡 공유

### 포스 오픈 절차

1. 카드리더기 전원 ON
2. 듀얼아이 전원 ON
3. 단말기 전원 ON
4. DUALL POS 앱 실행
5. 영업관리 → 영업전 클릭
6. 시재금 입력 (200,000원)

### 포스 오류 대응

| 증상 | 조치 |
|------|------|
| 연결 오류 | 단말기 OFF → ON |
| 로그인 오류 | 듀얼아이 OFF → ON |
| 앱 강제종료 | 앱 아이콘 길게 눌러 → 앱 정보 → 강제종료 → 재실행 |
| 현금영수증 화면 멈춤 | 서명패드에서 아무 번호 입력 → → 버튼 클릭 |

---

## 5. 미들 스텝 업무

- 중간 정산 실시
- 시재 확인
- 매장 청소 및 정리
- 하부장 재고 보충

### 택배 입고 시

1. 단톡방에 "택배 도착" 보고 + 사진
2. 거래명세서 확인 및 수량 파악
3. 수량 일치 여부 단톡 보고
4. 수량 불일치 시 해당 상품 별도 보관 + 보고
5. 거래명세서에 없는 상품은 **X 표시**

---

## 6. 마감 스텝 업무

### 마감 체크리스트

- [ ] 매장 청소
- [ ] 조명 OFF
- [ ] 배경 음악 OFF
- [ ] 토이카메라 충전 연결
- [ ] 보조배터리 충전 연결
- [ ] 포스 마감 처리
- [ ] 현금 보관
- [ ] 마감 후 매장 사진 촬영 → 단톡 공유
- [ ] 개인 음료/물 반드시 챙겨 퇴근

### 포스 마감 절차

1. 영업 종료 클릭
2. 시재 확인
3. 현금 분리 (시재금 / 매출금)
4. 영업 마감 클릭
5. 현금함 보관

---

## 7. 결제 매뉴얼

### 결제 멘트 (순서)

1. "결제 도와드리겠습니다"
2. 상품 수량 확인 (바코드 스캔 후 영수증 수량과 대조)
3. 쇼핑백 필요 여부 확인
4. 영수증 발급 여부 확인
5. "교환/환불은 7일 이내 영수증 지참 시 가능합니다"

### 결제 방식별

**카드 결제**
- 바코드 스캔 → 결제 진행

**현금 결제**
- 현금 금액 입력 → 잔돈 확인

**현금영수증**
- 개인: 전화번호 입력
- 사업자: 사업자번호 입력

**계좌이체**
- 이체 내역 + 영수증 사진 → 단톡방 공유

---

## 8. 환불 및 교환

### 기본 원칙

- **영수증 미지참 시 교환/환불 불가**
- **동일 상품 교환 불가** — 환불 후 재결제로 처리
- **부분 환불 불가**

### 카드 취소

- 결제 카드 지참 필수

### 현금영수증 취소

- 동일 번호 입력하여 취소

### 환불 전 확인사항

- 상품 이상 여부
- 영수증 금액
- 이벤트/할인 적용 여부

---

## 9. 불량 처리 절차

### 기본 원칙

- **불량 제품 서비스(무상 제공) 처리 절대 금지**

### CASE 1: 진열 중 발견

1. 불량 양식 작성
2. 사진 촬영 → 단톡방 업로드
3. 불량 양식 상품에 부착
4. 불량 상품 별도 보관 (카운터 뒤 → 가득 차면 창고)

### CASE 2: 판매 후 고객 불량 접수

**환불 처리 시:**
1. 구매 내역(영수증/카드내역) 확인
2. 환불 진행
3. 불량 양식 작성 + 보관

**교환 처리 시:**
1. 동일 상품으로 교환
2. 랜덤 상품인 경우 → "동일 캐릭터 나오지 않을 수 있음" 사전 안내
3. 재결제 없음
4. 불량 양식 작성 + 보관

## 11. 현금 관리

- **시재금:** 200,000원 (출근 시 확인)
- 만원/오천원/천원/동전 적정 비율 유지
- 지폐 부족 시 은행 환전 (주변 은행 위치 매니저 확인)

## 12. 주의사항

### 매장 관리

- 카운터 주변 항상 깔끔하게 유지
- 개인 음료/물 퇴근 시 반드시 폐기
- 매대 먼지 수시 닦기, 끈끈이 흔적 스티커 제거제 사용
- 하부장 적정량만 적재 (뒤로 넘어가지 않도록)
- 쓰레기장 이용 절차 확인 (팝업별 상이)

### 상품 관리

- 샘플 상품 수시 확인 (분실/훼손 체크)
- 샘플은 절대 판매 금지
- 상품 없는 경우 샘플도 매대에서 제거
- 비닐 뜯어진 상품 → OPP 재포장 후 판매 가능 (상품 자체 문제 없을 시)

### 고객 응대

- 결제 시 바코드 인식 후 수량 및 상품명 반드시 확인 → 오판매 방지
- 모르는 내용은 본사 담당자에게 연락 후 안내
- 분실 방지를 위해 매장을 자주 순회
