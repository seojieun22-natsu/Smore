# 🎪 팝업스토어 운영 가이드

> 팝업 첫 출근이라면 이 가이드를 처음부터 끝까지 읽어주세요!
> 이미 근무 중이라면, 필요한 섹션만 찾아보셔도 괜찮아요.

---

## 1️⃣ 하루의 흐름

팝업 하루는 **오픈 → 미들 → 마감** 3교대로 돌아가요.

> 💡 시프트 시간은 팝업마다 다를 수 있어요. 매니저 안내를 꼭 확인하세요!

| 시프트 | 시간 (예시) | 인원 |
|--------|------------|------|
| 오픈 | 08:30 ~ 13:30 / 09:00 ~ 14:00 | 2명 |
| 미들 | 11:00 ~ 16:00 | 1~2명 |
| 마감 | 15:30 ~ 20:30 | 2명 |

### 🍵 휴게 시간

- 근무 중 **30분** 휴게
- 휴게 전/후 **단톡방에 시간 보고** 필수

```
휴게 전: "00:00~00:00 휴게 다녀오겠습니다"
휴게 후: "00:00~00:00 휴게 다녀왔습니다"
```

---

## 2️⃣ 출퇴근은 이렇게

### 출근할 때

1. 단톡방에 **"출근했습니다"** 메시지
2. 유니폼 착용 + 스텝 목걸이 착용
3. 오픈 스텝이라면 → 매장 사진 촬영 후 단톡 공유

### 퇴근할 때

1. **퇴근 브리핑** 작성 (양식은 아래 참고)
2. 단톡방에 **"퇴근했습니다"** 메시지
3. 개인 물품(음료, 물 등) **반드시 챙겨가기**

### 근태 변경이 필요할 때

> ⚠️ 스케줄 이슈는 **매니저에게 먼저** 연락하세요!
> 매니저가 본사 담당자에게 전달합니다.

---

## 3️⃣ 복장 규정

| ✅ 필수 | ❌ 금지 |
|---------|---------|
| 유니폼(맨투맨) 착용 | 짧은 바지 |
| 스텝 목걸이 착용 | 슬리퍼 |
| 단정한 머리 및 복장 | 모자 + 마스크 동시 착용 |

---

## 4️⃣ 오픈 스텝 — 매장을 깨우는 사람

> 매장의 첫인상은 오픈 스텝이 만들어요!

### 체크리스트

- [ ] 포스기 전체 ON
- [ ] 조명 ON
- [ ] 배경 음악 ON
- [ ] AI 사진기 ON *(해당 시)*
- [ ] 토이카메라 샘플 배치 + 배터리 확인
- [ ] 매장 청소 (먼지, 쓰레기통 비닐 교체)
- [ ] 매대 상품 진열 상태 점검
- [ ] 하부장 재고 보충
- [ ] **샘플 상품 전체 확인** (분실/훼손 체크)
- [ ] 오픈 전 매장 사진 → 단톡 공유

### 포스 켜는 순서

```
카드리더기 ON → 듀얼아이 ON → 단말기 ON
→ DUALL POS 앱 실행 → 영업관리 → 영업전
→ 시재금 입력 (200,000원)
```

### 포스가 안 켜질 때

| 증상 | 이렇게 해보세요 |
|------|----------------|
| 연결 오류 | 단말기 OFF → ON |
| 로그인 안 됨 | 듀얼아이 OFF → ON |
| 앱이 꺼짐 | 앱 아이콘 길게 → 앱 정보 → 강제종료 → 재실행 |
| 서명패드 먹통 | 아무 번호 입력 → → 버튼 클릭 |

---

## 5️⃣ 미들 스텝 — 매장의 허리

### 기본 업무

- 중간 정산 + 시재 확인
- 매장 청소 및 정리
- 하부장 재고 보충

### 📦 택배가 왔을 때

> 택배 처리는 순서가 중요해요!

1. 단톡방에 **"택배 도착"** + 사진
2. 거래명세서 확인 + 수량 파악
3. 수량 일치 여부 단톡 보고
4. 불일치 상품은 별도 보관 + 보고
5. 거래명세서에 없는 상품 → **X 표시**

---

## 6️⃣ 마감 스텝 — 하루를 마무리하는 사람

### 체크리스트

- [ ] 매장 청소
- [ ] 조명 OFF + 배경 음악 OFF
- [ ] 토이카메라 충전 연결
- [ ] 보조배터리 충전 연결
- [ ] 포스 마감 처리
- [ ] 현금 보관
- [ ] **마감 후 매장 사진** → 단톡 공유
- [ ] 개인 음료/물 **반드시 폐기 또는 챙겨가기**

### 포스 끄는 순서

```
영업 종료 → 시재 확인 → 현금 분리 (시재금 / 매출금)
→ 영업 마감 → 현금함 보관
```

---

## 7️⃣ 결제할 때

### 결제 멘트 순서

> 자연스럽게 한 번씩 확인해주세요!

1. "결제 도와드리겠습니다"
2. 상품 수량 확인 *(바코드 스캔 → 영수증과 대조)*
3. "쇼핑백 필요하세요?"
4. "영수증 필요하세요?"
5. "교환/환불은 7일 이내 영수증 지참 시 가능합니다"

### 결제 방식별

| 방식 | 처리법 |
|------|--------|
| **카드** | 바코드 스캔 → 카드 결제 |
| **현금** | 현금 금액 입력 → 잔돈 확인 |
| **현금영수증** | 개인: 전화번호 / 사업자: 사업자번호 |
| **계좌이체** | 이체 내역 + 영수증 사진 → 단톡 공유 |

---

## 8️⃣ 환불 & 교환

### 기본 원칙 3가지

> 🚨 이 세 가지는 꼭 기억하세요!

1. **영수증 없으면** → 교환/환불 불가
2. **동일 상품 교환** → 불가 (환불 후 재결제)
3. **부분 환불** → 불가

### 처리 방법

| 상황 | 처리 |
|------|------|
| 카드 취소 | 결제 카드 지참 필수 |
| 현금영수증 취소 | 동일 번호 입력 |
| 다른 상품으로 교환 | 환불 → 재결제 |

### 환불 전 꼭 확인

- [ ] 상품 이상 여부
- [ ] 영수증 금액
- [ ] 이벤트/할인 적용 여부

---

## 9️⃣ 불량품이 나왔을 때

> ⛔ **기본 원칙: 불량 제품 서비스(무상 제공) 처리 절대 금지!**

### CASE 1: 진열하다가 발견

```
불량 양식 작성 → 사진 촬영 (단톡 업로드)
→ 양식을 상품에 부착 → 별도 보관
```

### CASE 2: 손님이 구매 후 불량 접수

**환불할 때:**
1. 구매 내역(영수증/카드내역) 확인
2. 환불 진행
3. 불량 양식 작성 + 보관

**교환할 때:**
1. 동일 상품으로 교환
2. 랜덤 상품이면 → **"동일 캐릭터 안 나올 수 있어요"** 꼭 안내
3. 재결제 없음
4. 불량 양식 작성 + 보관

## ⚠️ 이것만은 꼭!

### 매장 관리

- 카운터 주변 **항상 깔끔하게**
- 매대 먼지 수시 닦기 + 끈끈이 흔적은 스티커 제거제
- 하부장은 **적정량만** 적재 (뒤로 넘어가면 분실 위험)

### 상품 관리

- **샘플 = 판매 금지** (비닐 없는 전시용은 절대 판매 불가)
- 상품 없으면 샘플도 매대에서 제거
- 비닐 뜯어진 상품 → OPP 재포장 후 판매 가능

### 고객 응대

- 결제 시 **바코드 스캔 후 수량·상품명 반드시 확인** → 오판매 방지
- 모르는 건 본사 담당자에게 연락 후 안내
- 분실 방지 위해 **매장 자주 순회**
