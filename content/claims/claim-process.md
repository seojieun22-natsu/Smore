# 클레임 대응 프로세스

> 고객 클레임 발생 시 표준 대응 절차입니다.

## 🚨 클레임 대응 단계

### 1단계: 접수
- 고객 연락처, 구매 정보 확인
- 클레임 내용 상세 기록
- 사진/증거 자료 수집

### 2단계: 확인
- 제품 불량 여부 확인
- 배송 문제 여부 확인
- 책임 소재 판단

### 3단계: 처리
- 교환 / 환불 / 수리 결정
- 고객에게 처리 방안 안내
- 처리 완료 후 기록

### 4단계: 사후 관리
- 동일 클레임 재발 방지 조치
- 이카운트에 관련 내역 기록
